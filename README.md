# Skills Station

Welcome to Skills Station! This repository is a collection of useful skills for AI agents.

## Installed Skills

### 1. dokie-ai-ppt
- **Description**: AI presentation generator. Create professional HTML slides through conversation.
- **Source**: [myzy-ai/dokie-ai-ppt](https://github.com/myzy-ai/dokie-ai-ppt)
- **Status**: Third-party contribution

### 2. xmind
- **Description**: XMind file processing skill. Parse, create, and update XMind files.
- **Source**: [mitscherlich/skills](https://github.com/mitscherlich/skills)
- **Status**: Third-party contribution

### 3. arch-diagram-gen
- **Description**: Convert technical architecture, module, or workflow descriptions into professional, responsive HTML architecture diagrams.
- **Source**: Trae built-in skill
- **Status**: Created by Trae

### 4. html-to-image
- **Description**: Convert HTML files to PNG images using Puppeteer Core.
- **Source**: Trae built-in skill
- **Status**: Created by Trae

## How to Use

To use these skills, you can install them using the `npx skills add` command or by copying the skill directories to your project's `.agents/skills` or `.trae/skills` directory.

## Contributing

Feel free to contribute new skills or improve existing ones!
