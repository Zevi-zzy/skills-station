# Resume Generator Skill

## Description / 描述
A professional HTML resume generator skill. It creates a beautifully designed, print-ready resume based on user input, supporting Chinese character optimization and PDF export.

一个基于 HTML/CSS 的专业简历生成器。它能够根据用户输入生成排版精美、适合打印的简历，并针对中文字体和 PDF 导出进行了优化。

## Features / 功能特性
- **Professional Layout / 专业排版**: Classic two-column layout (Sidebar + Main Content). 经典的双栏布局（侧边栏+主内容）。
- **Print Ready / 打印优化**: Optimized for A4 paper size with `@media print`. 针对 A4 纸张优化的打印样式。
- **Chinese Support / 中文支持**: Font stack optimized for Chinese characters in PDF export. 针对 PDF 导出优化的中文字体栈。
- **Modular Design / 模块化设计**: Structured sections for Profile, Education, Skills, Experience, etc. 结构化的个人信息、教育、技能、经历模块。

## Usage / 使用方法
1. Provide your resume details (Name, Contact, Education, Experience, Skills). 提供你的简历详情。
2. The agent will generate an HTML file using the built-in professional template. Agent 将使用内置的专业模板生成 HTML 文件。
3. Open the generated HTML file in a browser. 在浏览器中打开生成的 HTML 文件。
4. Use `Print -> Save as PDF` to export your resume. 使用“打印 -> 另存为 PDF”导出简历。

## Template Source / 模板来源
Based on a high-quality resume design that has been verified in practice. 基于经过实践验证的高质量简历设计。
