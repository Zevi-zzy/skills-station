# Resume Generator Skill

## 1. 技能概述
这是一个基于 HTML/CSS 的专业简历生成器。它能够将用户的个人信息、经历和技能转换为排版精美、可直接打印的 HTML 简历。

## 2. 核心功能
- **专业排版**：采用左侧边栏 + 右侧主内容的经典双栏布局。
- **打印优化**：针对 A4 纸张进行 CSS 优化，确保打印输出完美（`@media print`）。
- **中文支持**：内置中文字体栈优化，解决 PDF 导出时的中文显示问题。
- **模块化设计**：支持个人信息、教育背景、技能标签、工作经历等模块。

## 3. 技术方案
- **输入**：用户提供简历内容的 JSON 数据或自然语言描述。
- **处理**：Agent 将内容填充到预置的 HTML 模板中。
- **输出**：生成一个独立的 HTML 文件。

## 4. 模板结构 (Template Structure)
模板包含以下关键部分：
1.  **全局样式**：定义配色变量（`--primary-color` 等）和字体。
2.  **左侧边栏**：
    *   个人信息（姓名、英文名）
    *   联系方式（电话、邮箱、地点）
    *   教育经历
    *   技能专长（分类标签）
    *   资质证书
3.  **右侧主内容**：
    *   工作经历（时间轴样式）
    *   项目经验（包含技术栈标签）

## 5. 打印优化策略
- 使用 `@page { size: A4; margin: 0; }` 消除默认页边距。
- 使用 `-webkit-print-color-adjust: exact !important;` 强制打印背景色。
- 隐藏非打印元素（如阴影）。

## 6. 使用方法
Agent 在接收到用户的简历信息后，应调用 `write_file` 工具，将填充好数据的 HTML 代码写入文件（例如 `resume.html`）。用户可以直接在浏览器打开该文件，并使用浏览器的“打印 -> 另存为 PDF”功能导出。
